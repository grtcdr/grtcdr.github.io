1. General settings

Disable the sourcing of the automatically created configuration file. 

config.load_autoconfig(False)

Save the session, so that I can pick up where I left off. 

c.auto_save.session = True

Customize the format of the tab title. 

c.tabs.title.format = '{index}: {current_title}'

Set the new tab and default pages to something custom: 

c.url.default_page = 'file:///home/grtcdr/.config/qutebrowser/startpage/index.html'
c.url.start_pages = 'file:///home/grtcdr/.config/qutebrowser/startpage/index.html'

2. Search engines

When performing a search, I could prefix my query, e.g. "System maintenance", with a search engine identifier, e.g. "aw". This will send the search query to the Arch Wiki instead of doing it with the DEFAULT search engine. A neat feature indeed! 

c.url.searchengines = {
	'aw': 'https://wiki.archlinux.org/index.php?search={}',
	'aur': 'https://aur.archlinux.org/packages/?O=0&K={}',
	'DEFAULT': 'https://www.google.com/search?q={}'
}
