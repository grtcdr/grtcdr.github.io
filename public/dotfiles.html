dotfiles

Preface

The documentation found herein follows the literate programming principle.

It's only possible thanks to the Emacs text editor, and the Org Mode ecosystem.

The documentation does not cover all of my configurations. Hence, I invite you to browse the repository where the rest of my dotfiles live.

Please note that a small but significant portion of my configurations, is a mold of what I've read in a blog, seen in a demonstration, or just adapted from someone else's dotfiles.

  • sway

    A concoction of functionalism and elegance, making use of the bare minimum of software so as to achieve a coherent desktop environment without one being present.

    • colorschemes
    • swaylock
    • mako

  • zsh

    A short zsh configuration targeting simplicity.

  • doch

    A shell script that lists and opens HTML/PDF files found within my org-directory.

  • startw

    A shell script that intelligently starts sway.

  • mus

    A shell script that automates the starting and stopping of mpd and ncmpcpp.
