| Directory | Contents                          |
|-----------+-----------------------------------|
| =rc=        | Literate configuration files      |
| =sh=        | Literate shell scripts            |
| =pro=       | List of projects I've worked on   |
| =data=      | Data files exposed to the website |
| =img=       | Images used throught the website  |
| =css=       | Stylesheets                       |
