Depot

The documentation found herein follows the literate programming principle, it does not cover all of my configurations, so you are invited to browse the repository where the rest of my dotfiles live.

This project was made possible thanks to the Emacs text editor, and the Org Mode ecosystem - this is not to say that a literate programming implementation is exclusive to these two, but I happen to like their integration and extensibility.

Please note that a small portion of my configurations may contain something adapted from someone's [a] blog, [b] video demonstration or [c] dotfiles.
